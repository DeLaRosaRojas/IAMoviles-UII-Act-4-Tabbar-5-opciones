#  Parisina - De La Rosa 6I ---- Prompt modificado por la AI :P

Proyecto desarrollado en **Flutter & Dart** para la tienda de telas y costura "Parisina - De La Rosa 6I". Esta aplicación móvil presenta un catálogo visual dividido en categorías clave del negocio.

##  Especificaciones de Diseño

La interfaz ha sido diseñada siguiendo una identidad corporativa vibrante y funcional:

* **AppBar:** Fondo Rojo con tipografía en Blanco.
* **Acentos:** Subtonos amarillos, negros y grises para elementos de navegación y fondos.
* **Navegación:** TabBar inferior (BottomNavigationBar) con 5 secciones dinámicas.

##  Requerimientos Técnicos

Este proyecto fue generado siguiendo este prompt de desarrollo:

> "Generar una aplicación en Flutter para el negocio **Parisina - De La Rosa 6I**. Implementar un AppBar rojo con letras blancas y una paleta con amarillos/negros/grises. Crear un TabBar inferior con 5 opciones (Telas, Hilos, Hogar, Vestir, Maquinas) incluyendo iconos representativos. Cada página debe mostrar una imagen centrada desde una URL específica de GitHub y el código debe estar separado en `main.dart` y archivos independientes para cada tab."
